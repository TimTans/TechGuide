# Deployment QA Checklist

## Pre-Deployment
- [ ] All tests passing
- [ ] Code coverage > 80%
- [ ] No critical bugs
- [ ] Build successful

## Deployment
- [ ] Deploy to staging
- [ ] Run smoke tests
- [ ] Deploy to production
- [ ] Monitor logs

## Post-Deployment
- [ ] Verify homepage loads
- [ ] Test authentication
- [ ] Check error rates
- [ ] Monitor performance
